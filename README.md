# MyBOT — AI Coding Assistant

An AI-powered coding assistant built with Flask and Groq's LLaMA model. 
Ask anything about code — debugging, explanations, reviews, or concepts.

## Features
- Real-time AI responses powered by LLaMA 3.3 70B
- Clean dark theme interface built for developers
- Code syntax highlighting with copy button
- Conversation memory within session
- Clear history option

## Tech Stack
- **Backend:** Python, Flask
- **AI:** Groq API (LLaMA 3.3 70B)
- **Frontend:** HTML, CSS, JavaScript
- **Deployment:** Render

## Setup

1. Clone the repository
   git clone https://github.com/AkshitaSharma211/API-LEARNING.git

2. Install dependencies
   pip install -r requirements.txt

3. Add your Groq API key in .env file
   GROQ_API_KEY=your_key_here

4. Run the app
   python3 app.py

5. Open http://127.0.0.1:5000 in browser
